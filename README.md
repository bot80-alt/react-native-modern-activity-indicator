A modern and customizable **activity indicator component for React Native**.

Provides smooth, visually appealing loading animations that can be easily integrated into
React Native applications. Designed with simplicity, performance, and modern UI aesthetics
in mind.

---

## Overview

`react-native-modern-activity-indicator` is a lightweight UI component that helps you
indicate loading states during asynchronous operations such as API calls, screen transitions,
and background tasks.

The indicator is fully customizable and works seamlessly across **Android** and **iOS**.
